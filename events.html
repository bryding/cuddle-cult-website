Google Calendar Integration

This page synchronizes with your Google Calendar. To configure it:

  1. Create a public Google Calendar for your camp events
  2. Get your Calendar ID from Google Calendar Settings
  3. Create a Google API Key with access to the Calendar API
  4. Edit public/js/calendar.js and add your Calendar ID and API Key

Once configured, this page will automatically display events from your Google Calendar organized by day of the week.
